# Zoeken in het Archief

Deze pagina legt uit hoe je effectief kunt zoeken in het KNA Historie archief.

## Globale Zoekfunctie

### Zoekbalk Gebruiken

De zoekbalk vind je bovenaan elke pagina:

1. **Klik** in het zoekveld (of druk op `/`)
2. **Typ** je zoekterm
3. **Druk** op Enter of klik op het vergrootglas icoon
4. **Bekijk** de resultaten

### Geavanceerd Zoeken

#### Meerdere Zoektermen

Gebruik spaties om meerdere termen te combineren:

```
Annie Get Your Gun 2015
```

Vindt: Voorstelling "Annie Get Your Gun" uit 2015

#### Exacte Zinnen

Gebruik aanhalingstekens voor exacte matches:

```
"de burgemeester"
```

Vindt alleen: Exacte rol "de burgemeester"

#### Wildcards

Gebruik `*` voor onbekende delen:

```
Jan*
```

Vindt: Jan, Jansen, Janssen, etc.

## Zoekresultaten Verfijnen

### Te Veel Resultaten?

Als je zoekactie te veel resultaten oplevert:

1. **Voeg meer specifieke termen toe**
   ```
   Voor:  "Jan"
   Na:    "Jan de Vries acteur 2015"
   ```

2. **Gebruik filters**
   - Beperk tot specifiek jaar
   - Kies een voorstellingstype
   - Selecteer een rol

3. **Zoek in specifieke sectie**
   - Ga naar Leden/Voorstellingen/Tijdlijn
   - Zoek daar specifiek

### Te Weinig Resultaten?

Als je niets vindt:

1. **Controleer spelling**
   - Let op typfouten
   - Probeer alternatieve spelling
   - Gebruik wildcards (*)

2. **Maak zoekterm breder**
   ```
   Voor:  "Jan de Vries acteur Annie Get Your Gun 2015"
   Na:    "Jan Vries 2015"
   ```

3. **Zoek op andere velden**
   - Voornaam in plaats van achternaam
   - Jaar in plaats van titel
   - Regisseur in plaats van acteur

## Geavanceerde Zoekopties

### Boolean Operators

Voor ervaren gebruikers:

**AND** - Beide termen moeten voorkomen
```
Jan AND Vries
```

**OR** - Een van beide termen
```
Jan OR Piet
```

**NOT** - Sluit term uit
```
voorstelling NOT musical
```

**Groeperen met haakjes**
```
(Jan OR Piet) AND 2015
```

### Veld-specifiek Zoeken

Zoek in specifieke velden:

```
voornaam:Jan
achternaam:Vries
jaar:2015
type:musical
regisseur:Piet
rol:hoofdrol
```

Combineer meerdere velden:
```
voornaam:Jan jaar:2015 type:musical
```
