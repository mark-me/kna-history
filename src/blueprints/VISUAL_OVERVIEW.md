### Status Colors

- **Success**: Green (#10B981)

  - Data loaded successfully
  - Thumbnails generated

  - Validation failed
  - Load error
  - Missing files

  - Non-critical warnings
  - Data quality issues

  - File selected
  - Progress indicators
  - Statistics

- **Primary Actions**: Blue buttons
- **Danger Actions**: Red buttons
- **Secondary Actions**: Gray buttons
- **Disabled State**: Light gray
